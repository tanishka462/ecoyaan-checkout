# Ecoyaan Checkout

## Tech Stack
- Next.js
- React
- TailwindCSS
- React Hook Form

## Architecture
The application uses Next.js App Router with component-based architecture.

Main pages:
- Home
- Address
- Payment
- Success

Cart state is managed using React Context.

## Features
- Checkout flow
- Address form validation
- Phone number input
- Local storage for saved address
- Responsive UI

## Run Locally

Clone repository

git clone https://github.com/tanishka462/ecoyaan-checkout

Install dependencies

npm install

Run development server

npm run dev

Open

http://localhost:3000
